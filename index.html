📌 Programas WMS

Programa 6173

Função:

O programa 6173 é utilizado para consulta de cadastro de itens, permitindo verificar informações como item depositante, descrição e altura.

Categoria: Consulta de itens

Sistema: WMS

Programa WMS10021

Função:

O programa WMS10021 é utilizado para cadastro completo de itens, permitindo alterar controles gerais do item.

Categoria: Cadastro de itens

Sistema: WMS
